## Project Overview

This project focuses on transforming raw business data into a comprehensive Business Intelligence dashboard using Microsoft Power BI.

The objective was to clean, model, and visualize business data in a structured way to support decision-making and improve reporting accuracy through interactive dashboards.

---

## Key Responsibilities

- Cleaned and transformed raw business data using **Power Query**
- Built calculated columns and custom measures using **DAX**
- Designed an optimized **data model** to ensure accurate KPI calculations
- Developed interactive dashboards with slicers and drill-through functionality
- Created custom KPIs and performance metrics to enhance reporting clarity

---

## Key Outcomes

- Delivered a comprehensive BI report demonstrating:
  - Strong data transformation workflow
  - Accurate KPI development
  - Effective data modeling
  - Clear data storytelling

- Improved dashboard usability through:
  - Drill-through reports
  - Dynamic slicers
  - Interactive filtering
